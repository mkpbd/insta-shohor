What is even loop ?

JavaScript has a runtime model base on even loop Which is responsible for executing code collecting and processing events executing The event loop is the secret behind javascript asynchronous programming.
JS executes all operations on a single thread but using a few smart data structures it gives us the multi-threading
1. The call stack is responsible for keeping track of all the operation in line to be executed whenever a function is finished, it is popped from the stack
2. The even queue is responsible for sending new functions to the stack for processing.
it follow the queue data structure to maintain the correct sequence all operation should be sent for execution whenever an asynchronous function is called it is sent to a browser API, These are APIs built into the browser base on command receive from the stack the API starts its own single-threaded operation example setTimeout() function , clearTimeout()
If a function Tasks a long time to execute we can not interact with the web browser during the function's execution because the page hangs. The event loop is a constantly runnig process that monition both the callback queue and the call stack If the call stack is not empty the event loop waits until it is empty and places the next function from the callback queue to the call stack, if the callback queue is empty nothing will happen

What is difference localstorage between sessionStorage ?

LocalStorage and SessionStorage are almost identical and have the same API, The difference is that with sessionStorage, the data is persisted only until the window or tab is closed. In the LocalStorage the data is persisted until the user manually clears the browser cache or until your web page clears the data. but the syntax for sessionStorage are same we can create, read, and update key/value pairs in localStorage we can create entries for the localStorage object by using the setItem() method. The setItem() method takes two arguments the key and corresponding value to read entries use the getItem() the getItem() method take one arguments deleting and clearing Entries we can delete an entry with the removeItem() the removeItem() method takes one argument which will be a key of the localStorage object
localStorage use for long term use on the other hand SessionStorage use when we need to store somthing that changes or somthing temporary
